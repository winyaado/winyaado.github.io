# ぬこ ポートフォリオ

これは、レトロなWindows 3.1テーマの個人ポートフォリオウェブサイトです。

## 機能

*   **プロフィール**: 「ぬこ」の自己紹介。
*   **スキル**: 技術スキルと経験の概要。
*   **活動内容**: 過去のプロジェクトへのリンク。
*   **連絡先**: 連絡先情報。
*   **フォト**: シンプルな写真ギャラリー。
*   **おみくじ**: 日本のおみくじゲーム。

## 実行方法

これは静的なウェブサイトです。`index.html`をウェブブラウザで開くだけでポートフォリオを閲覧できます。
